# prettier-plugin-wgsl

This project is inspired by [prettier-plugin-glsl](https://github.com/NaridaL/glsl-language-toolkit) and also uses [Chevotain](https://chevrotain.io/docs/) for parsing.

Disclaimer: This project is very early stage and developed mostly as an experiment to learn how to use claude for bigger projects.

### Installation

```sh
npm install --save-dev prettier-plugin-wgsl
```

### Usage

Add `prettier-plugin-wgsl` to your Prettier configuration:

```json
{
  "plugins": ["prettier-plugin-wgsl"]
}
```
